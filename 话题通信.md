# 话题通信
## 创建功能包
- 创建工作空间
```C
mkdir ROS2_WS
cd ROS2_WS
mkdir src
```
- 进入工作空间下的src目录
```C
cd ROS2_WS/src
```
- 创建功能包
```C
ros2 pkg create topicPKG --build-type ament_cmake --dependencies rclcpp --node-name pub
```
